# @hope-ui/palettes

Hope UI color palettes built on top of TailwindCSS colors.

## Installation

```bash
npm install @hope-ui/palettes
# or
yarn add @hope-ui/palettes
# or
pnpm add @hope-ui/palettes
```

## Documentation

For full documentation, visit [hope-ui.com](https://hope-ui.com/).
