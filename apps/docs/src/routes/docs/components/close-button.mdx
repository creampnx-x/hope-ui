# CloseButton

A button with a close icon.

## Import

```tsx
import { CloseButton } from "@hope-ui/core";
```

## Usage

<Preview>
  <CloseButton />
</Preview>

```tsx
<CloseButton />
```

### Sizes

Use the `size` prop to change the size of the `CloseButton`.

<Preview>
  <HStack spacing={4}>
    <CloseButton size="sm" />
    <CloseButton size="md" />
    <CloseButton size="lg" />
  </HStack>
</Preview>

```tsx
<HStack spacing={4}>
  <CloseButton size="sm" />
  <CloseButton size="md" />
  <CloseButton size="lg" />
</HStack>
```

### Custom icon

To use a custom icon, pass it as `children` of the `CloseButton`.

<Preview>
  <CloseButton>
    <CrossCircledIcon />
  </CloseButton>
</Preview>

```tsx
<CloseButton>
  <CrossCircledIcon />
</CloseButton>
```

If you want to create your own icon components check out Hope UI [Icon](/docs/components/icon) documentation.

## Theming

Use the `CloseButtonTheme` type to override `CloseButton` styles and default props when extending Hope UI theme.

```tsx
import { extendTheme, CloseButtonTheme } from "@hope-ui/core";

const theme = extendTheme({
  components: {
    CloseButton: {
      // ...overrides
    } as CloseButtonTheme,
  },
});
```

### Component parts

| Name | Static selector         | Description  |
| ---- | ----------------------- | ------------ |
| root | `hope-CloseButton-root` | Root element |

## Props

| Name       | Type                   | Description                        | Default value |
| ---------- | ---------------------- | ---------------------------------- | ------------- |
| aria-label | `string`               | A label that describes the button. | "Close"       |
| size       | `"sm" \| "md" \| "lg"` | The size of the button.            | md            |
| children   | `JSX.Element`          | The icon to be used in the button. |               |
