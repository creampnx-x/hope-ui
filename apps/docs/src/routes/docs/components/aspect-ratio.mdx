# AspectRatio

Display images, videos, maps and other media with a consistent width/height ratio.

## Import

```ts
import { AspectRatio } from "@hope-ui/core";
```

## Usage

Use the `ratio` prop to define a consistent width/height ratio.

<Preview>
  <AspectRatio ratio={9 / 16} w="full" maxW="200px">
    <Image src="https://bit.ly/3TJwRZR" alt="Monkey D. Luffy" fit="cover" />
  </AspectRatio>
</Preview>

```tsx
<AspectRatio ratio={9 / 16} w="full" maxW="200px">
  <Image src="https://bit.ly/3TJwRZR" alt="Monkey D. Luffy" fit="cover" />
</AspectRatio>
```

### Embed a video

<Preview>
  <AspectRatio ratio={1} w="full" maxW="560px">
    <iframe
      title="one piece opening 1"
      src="https://www.youtube.com/embed/HRaoYuRKBaA"
      allowfullscreen
    />
  </AspectRatio>
</Preview>

```tsx
// This video will have equal sides
<AspectRatio ratio={1} w="full" maxW="560px">
  <iframe
    title="one piece opening 1"
    src="https://www.youtube.com/embed/HRaoYuRKBaA"
    allowfullscreen
  />
</AspectRatio>
```

### Embed a map

<Preview>
  <AspectRatio ratio={16 / 9} w="full">
    <iframe title="Yosemite National Park" src="https://bit.ly/3KKbOm9" />
  </AspectRatio>
</Preview>

```tsx
<AspectRatio ratio={16 / 9} w="full">
  <iframe title="Yosemite National Park" src="https://bit.ly/3KKbOm9" />
</AspectRatio>
```

## Props

| Name  | Type                      | Description                                  | Default value |
| ----- | ------------------------- | -------------------------------------------- | ------------- |
| ratio | `ResponsiveValue<number>` | The aspect ratio of the Box (ex: 16/9, 4/3). | 4/3           |
