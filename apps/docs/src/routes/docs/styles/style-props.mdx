# Style props

Define component's style simply by passing props.

## Reference table

The following tables shows a list of every style props and the CSS properties their map to.

### Margin and padding

```tsx
import { Box } from "@hope-ui/core";

// Refers to the value of `theme.vars.space["2"]`
<Box m={2}>Tomato</Box>

// You can also use custom values
<Box maxW="960px" mx="auto" />

// Sets margin `8px` on all viewports and `12px` from the first breakpoint and up
<Box m={[2, 3]} />
```

| Prop                  | CSS Property                                  | Theme scale |
| --------------------- | --------------------------------------------- | ----------- |
| `m`, `margin`         | `margin`                                      | `space`     |
| `mt`, `marginTop`     | `margin-top`                                  | `space`     |
| `mr`, `marginRight`   | `margin-right`                                | `space`     |
| `me`, `marginEnd`     | `margin-inline-end`                           | `space`     |
| `mb`, `marginBottom`  | `margin-bottom`                               | `space`     |
| `ml`, `marginLeft`    | `margin-left`                                 | `space`     |
| `ms`, `marginStart`   | `margin-inline-start`                         | `space`     |
| `mx`, `marginX`       | `margin-inline-start` + `margin-inline-end`   | `space`     |
| `my`, `marginY`       | `margin-top` + `margin-bottom`                | `space`     |
| `p`, `padding`        | `padding`                                     | `space`     |

### Pseudo selector

<Button
  colorScheme="primary"
  _hover={{
    background: "white",
    color: "primary.500",
  }}

  Hover me
</Button>

  role="group"
>
  <Box
    _hover={{ fontWeight: 'semibold' }}
    _groupHover={{ color: 'tomato' }}
  >
  </Box>
</Box>

// add ::before pseudo element
// Note: the content value needs an extra set of quotes!
<Box
  _before={{ content: '"🙂"', display: 'inline-block', mr: '5px' }}
>
  A pseudo element
</Box>

## The `as` prop

All Hope UI components are polymorphic, meaning you can use the `as` prop to change the rendered element.

For example, say you are using the `Button` component, and you need to make it a link instead. You can compose `a` and `Button` like below:

<Preview>
  <Button as="a" target="_blank" href="https://solidjs.com">
    Go to SolidJS website
  </Button>
</Preview>

```tsx
<Button as="a" target="_blank" href="https://solidjs.com">
  Go to SolidJS website
</Button>
```

And it works with SolidJS components too:

<Preview>
  <Button as={Link} href="/">
    Back to home
  </Button>
</Preview>

```tsx
import { Link } from "@solidjs/router";

function App() {
  return (
    <Button as={Link} href="/">
      Back to home
    </Button>
  );
}
```
